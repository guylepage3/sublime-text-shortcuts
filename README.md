# sublime-text-shortcuts

| Key Command      | Description      |
| :----------------- | :---------------------------------------------------------------- |
| **Groups**      |       |
| ⌘K ⌘↑                | Move File to New Group — Up      |
| ⌘K ⇧⌘↑             | New Group      |
| ⌘K ⇧⌘↓             | Close Group      |
| ⌘K ⇧⌘←             | Move File to Group — Previous      |
| ⌃⇧1                      | Move File to Group — Group 1      |
| ⌃⇧2                     | Move File to Group — Group 2      |
| **Sidebar**      |       |
| ⌘K ⌘B                 | Open and close sidebar      |
